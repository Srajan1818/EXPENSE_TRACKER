💰 Expense Tracker

A simple and responsive **Expense Tracker App** built with **HTML**, **CSS**, and **JavaScript** that helps you manage your daily expenses by tracking your income and expenditure in real time.

---

## ✨ Features

- ➕ Add income and expense transactions
- 🗑️ Delete any transaction instantly
- 💵 Displays total balance in real time
- 💾 Transactions saved to localStorage
---

## 🎯 Project Purpose

This project was created to strengthen the following concepts:

- 🔄 Dynamically adding and removing DOM elements
- 🧮 Performing real time balance calculations using JavaScript
- 📱 Building a clean and fully responsive financial UI

---

## 🛠️ Technologies Used

- **HTML5** – Structure and markup
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6+)** – Transaction logic, balance calculation and localStorage

## 📌 How To Use

1. Enter a **transaction name** in the text field.
2. Enter the **amount** (positive for income)
3. Click **Add Transaction** button.
4. View your **total balance**, **income** and **expense** updated instantly.
5. Click the delete button to **delete** any transaction.
